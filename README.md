## 🌟 Overview

**BrightSteps** is an interactive, gamified learning environment tailored to support children with autism and ADHD. The platform bridges the gap between students, parents, and educators by providing:
- A distraction-free **Student Dashboard** filled with curated routines, resources, and games.
- A **Parent Portal** for tracking progress and managing access.
- A **Teacher Dashboard** for monitoring performance and adjusting curriculums.

## ✨ Features

- 🎮 **Gamified Learning**: Interactive tasks and games designed to retain focus and encourage progress.
- 📚 **Resource Manager**: A beautiful, intuitive interface for students to access tailored videos, PDFs, and links.
- 🔒 **Parental Controls**: Secure PIN-protected access for parents to review detailed analytics.
- 👩‍🏫 **Teacher Hub**: Tools for educators to assign resources, track student streaks, and analyze learning patterns.
- 🎨 **Accessibility-First Design**: Custom color palettes, clear typography, and simplified navigation suited for neurodivergent learners.

## 🛠️ Tech Stack

- **Frontend**: React, React Router v7, Lucide Icons
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Authentication**: JWT & Google OAuth

## 🚀 Quick Start (Local Development)

### 1. Backend Setup
1. Open a terminal and navigate to the `server` folder:
   ```bash
   cd server
   ```
2. Create a `.env` file based on `.env.example` and fill in the necessary API keys and database URIs.
3. Install dependencies and start the server:
   ```bash
   npm install
   node server.js
   ```

### 2. Frontend Setup
1. Open a new terminal and navigate to the `client` folder:
   ```bash
   cd client
   ```
2. Install dependencies and start the React app:
   ```bash
   npm install
   npm start
   ```
