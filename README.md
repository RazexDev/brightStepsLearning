# BrightSteps Learning Platform

A MERN stack application for autism/ADHD support and teacher-parent communication.

## Quick Start (Local Development)

### 1. Backend Setup
1. Open a terminal in the `server` folder.
2. Create a `.env` file based on `.env.example`.
3. Run:
   ```bash
   npm install
   node server.js
   ```

### 2. Frontend Setup
1. Open a new terminal in the `client` folder.
2. Run:
   ```bash
   npm install
   npm start
   ```

## Tech Stack
- **Frontend:** React, Lucide Icons, React Router 7
- **Backend:** Node.js, Express, MongoDB
- **Auth:** JWT + Google OAuth
